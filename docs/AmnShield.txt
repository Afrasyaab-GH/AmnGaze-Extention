🛡️ AmnShield Web Extension — Complete Feature & Development Plan
1. Overview
AmnShield Web Extension is a comprehensive digital protection system that enforces spiritual discipline, focus, and safe browsing.
It provides:
• Islamic Mode (Flexible + Full Protection)
• Work Mode
• School Mode
• Personal Protection Mode
• Time-based and timetable-based blocking
• Default Tab Integration for Islamic reminders, Qur’an, Hadith, du‘ā, and motivational content
The extension is cross-platform ready (web first), with cloud sync for unlock codes, schedules, and tamper detection.
2. Modes of Protection
2.1 Islamic Mode — Two Layers
2.1.1 Flexible Islamic Protection (Dynamic Schedule)
Characteristics:
• Non-editable by the user
• Auto-blocks based on: 
• Daily prayer times (1 hour before → 1 hour after each prayer)
• Sleep period (Isha → Fajr, ~9–10 hours)
• Jumu‘ah (Friday) — all day
• Ramadan — all week during fasting hours
• Sacred Months — Muharram, Dhul-Qi’dah, Dhul-Hijjah, Rajab (full-month protection)
• Special Islamic Days — Laylat al-Qadr, Eid, Ashura, and other significant dates
Default Tab displays:
• Prayer countdowns
• Qur’an verses
• Du‘ā
• Reflection prompts
• Sacred month notifications

2.1.2 Full Protection (Strict Lock)
Characteristics:
• Fully blocks all digital content except essential Islamic resources and AmnShield Default Tab
• Non-bypassable except via AmnShield Unlock Code
• Tamper detection triggers full lockdown if override is attempted
JSON Example:
{ "mode": "islamic", "variant": "full_protection", "lockState": "strict", "requiresUnlockCode": true, "blockedCategories": ["all"], "whitelist": ["amn-shield-tab", "quran.com", "sunnah.com"] } 
2.2 Work Mode
• Time-based blocking during work hours
• Whitelist for productivity/business websites
• Optional focus sessions (Pomodoro or deep-focus timer)
• Syncs with Default Tab for motivation and reminders
2.3 School Mode
• Timetable-based blocking (select days and school hours)
• Blocks entertainment, social media, gaming
• Optional timed study breaks
• Default Tab displays study motivation and goal tracking
2.4 Personal Protection Mode
• Custom digital discipline schedule
• Blocks distractions during selected hours
• Optional weekend relax periods
• Syncs with mobile app notifications
