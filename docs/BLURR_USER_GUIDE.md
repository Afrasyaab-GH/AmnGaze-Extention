# AmnShield Blurr Feature - Quick Start Guide

## What is Content Blur?

The Content Blur feature automatically blurs potentially inappropriate images, videos, and other media on web pages, giving you an extra layer of protection beyond site blocking.

---

## Quick Access

### From Popup (Fastest)

1. Click the AmnShield icon in your browser toolbar
2. Look for "🔍 Content Blur" toggle
3. Click the switch to turn blur ON or OFF
4. Blur applies immediately to all open tabs

### From Settings (Full Control)

1. Click the AmnShield icon
2. Click "⚙️ Settings" button
3. Scroll to "🔍 Content Blur Settings"
4. Configure all options

---

## Basic Usage

### Enable Blur Globally

**Popup Method:**

- Click extension icon → Toggle "Content Blur" ON

**Settings Method:**

- Settings → Content Blur Settings → Enable "Enable Content Blur"

### Keyboard Shortcuts

#### Alt+L - Quick Toggle

- Press `Alt+L` anywhere to turn blur ON/OFF
- Works on current tab only
- Perfect for quick privacy control

#### Alt+K - Selective Unblur

1. Hover your mouse over an image/video
2. Press `Alt+K`
3. The element will unblur (or reblur if already unblurred)
4. Great for checking blurred content safely

## Configuration Options

### What Can Be Blurred?

- ✅ **Images** - All img tags
- ✅ **Videos** - Video elements
- ✅ **iFrames** - Embedded content
- ✅ **Background Images** - CSS backgrounds
- ✅ **Adjustable Intensity** - 1-50px blur

### Visual Effects

- **Blur Amount**: Adjust with slider (default: 20px)
- **Grayscale**: Make blurred content black & white

---

## Whitelist (Domain Exceptions)

### When to Use

Add domains you trust so blur never applies there:

- Work/school sites you need clear
- Family-friendly sites like news/education
- Sites where blur interferes with functionality

### How to Add

1. Go to Settings
2. Find "Content Blur Settings"
3. Under "Whitelist Domains" type domain name
4. Click "Add" button

**Example:** `wikipedia.org` or `khanacademy.org`

### How to Remove

- Click the × next to any domain in the list

## Troubleshooting

### Blur Not Working?

1. Check if Content Blur is enabled in popup
2. Make sure current site isn't in whitelist
3. Reload the page (Ctrl+R or Cmd+R)
4. Try different blur intensity

### Some Images Not Blurring?

- Some sites load images dynamically
- Try refreshing the page
- Increase blur intensity
- Report issue if persistent

### Keyboard Shortcuts Not Working?

- Close the popup first (shortcuts don't work when popup is open)
- Make sure you're on a regular web page (not chrome:// or about: pages)
- Check if another extension uses the same shortcut

### Blur Too Weak/Strong?

- Adjust "Blur Intensity" slider in Settings
- Default is 20px
- Try 30-40px for stronger blur
- Try 10-15px for lighter blur

## FAQ

**Q: Does blur affect page load speed?**
A: No, blur is CSS-based and has minimal impact.

**Q: Can I blur only images but not videos?**
A: Yes! Uncheck "Blur Videos" in settings.

**Q: Will blur work on all sites?**
A: Yes, except sites you add to whitelist.

**Q: Can I have different blur levels for different sites?**
A: Not yet, but it's planned for future updates.

**Q: Does blur work in incognito/private mode?**
A: Yes, if you enable the extension in incognito mode.

**Q: Will my settings sync across devices?**
A: Yes, if you're signed into your browser account.
